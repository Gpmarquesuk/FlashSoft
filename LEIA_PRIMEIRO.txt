🚀 COMO USAR O INTERVIEW ASSISTANT:

1. Abra a UI (use demo_ui.bat ou comando PowerShell)

2. Carregue os arquivos de exemplo:
   ├─ Resume: examples\manual_inputs\Resume-Gustavo-Marques Sep25.docx
   ├─ Job Description: examples\manual_inputs\Technical Application &amp_ Integration Specialist .pdf
   └─ Audio (opcional): examples\manual_inputs\captura_wasapi_autodetect.wav

3. Use os atalhos:
   ├─ Alt+S: Iniciar sessão
   ├─ Alt+G: Gerar resposta
   └─ Alt+E: Encerrar sessão

4. O overlay exibirá respostas curtas (≤45 palavras) com talking points
